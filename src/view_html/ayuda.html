BitPraes

¿Cómo usar BIT·PRAES?

  1. Inserta una imagen en el área indicada o arrástrala desde tu sistema.
  2. Haz clic en INICIAR BÚSQUEDA para analizar la imagen y detectar rostros.
  3. Visualiza los rostros encontrados (máximo 3) en el panel izquierdo.
  4. Consulta el mapa para ubicar los puntos donde se detectaron coincidencias.
  5. Revisa los resultados en la tabla de coordenadas y cámaras.

Funciones disponibles:

  • REANALIZAR IMAGEN: Si hiciste cambios o la búsqueda fue incorrecta.
  • IMPRIMIR RESULTADOS: Abre el gestor de impresión para exportar tu análisis.
  • EXPORTAR RESULTADOS: Guarda el resultado como archivo .PDF o .CSV.
  • AYUDA / INSTRUCCIONES: Te lleva a esta ventana con instrucciones.

Notas importantes:

  • Formatos permitidos: .JPG, .PNG.
  • Se recomienda usar imágenes nítidas y bien iluminadas.
  • Este sistema es experimental. Verifica manualmente la información antes de usarla como evidencia.
